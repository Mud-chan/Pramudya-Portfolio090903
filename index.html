Endog Delivery Platformer Game

The game is a platformer with narrative elements, combining physical obstacle-based challenges with a deep story. The genre was chosen to provide an interactive and emotional gaming experience, where players not only complete challenges but also connect with the main character's journey.

The initial display of the game contains several button icons in the form of Play Game and Exit.

When the user presses the Play Game button, it will display the game loading page and how to play the game.

The following is a screenshot of a page showing how the game system is run (Gameplay).

When the user clicks the button in the top right corner (resume), the resume (continue), restart (restart the game) and home (return to the home page) menu options will appear.

The Game Over page appears when the character has lost the number of eggs delivered (marked by the number of eggs in the left corner or the character falls into a ravine (water swamp).

The Finish Game page appears when the character reaches the finish line.

Eggrade Grading Telur

Eggrade is a website that classifies chicken egg quality based on color, thickness, cracks, and weight. Eggrade is integrated with IoT and uses an intelligent system based on CNN and KNN and the python flask framework. Even without the IoT tool, this website can still run normally but cannot classify the weight of eggs because it requires a tool. Here are the features and how it works:

On the landing page there is some information in the form of, What features are available on the Eggrade website and what technology is applied to the website such as information on the smart system used and what percentage of accuracy it has.

On the main page there is an image upload menu for those who do not use the tool, there must be 2 chicken egg images, namely the right and left sides. then on the left there is a choice of classification based on certain categories and the choice of methods used is CNN and KNN.

For IoT users, the image will be automatically captured when an egg is detected in the tool. The image will be sent to the web automatically and if there are already 2 images, it will automatically run the classification.

The classification results will display details of each image and are combined then the final result will appear in the form of Grade A to C and there is also Reject if the egg quality is poor.

In addition to the final results, there are also details on each image processing from cropped image to completion. Why should there be image processing details? because this project is not only for Rizky farm but also for learning about image processing on CNN and KNN.

Here are photos of the tools and output results on the LCD

The IoT components used are, Loadcell as a weight sensor, ESP as a microcontroller, ESP32-CAM as a camera that takes pictures, LCD as an output display, Relay Module as a light sensor to turn on the machine.
